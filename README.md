# Thoughtbox

Thoughtbox is an app that accepts user input of up to 1200 characters long and submits them to a database. Each input is called a thought. 

This is a web app designed for those who want to collect website visitors' anonymous thoughts. The most practical use for it is as a suggestions box for entities who would like anonymous feedback from visitors to their site. A less practical but no less interesting use for it is as a place to throw your OWN thoughts for YOURSELF to read later. Tweet... to your future database-checking self!

The app is mobile-ready. Specify your database as an enivironment variable to make the app your own.

Find the app deployed [here](https://thotbox.herokuapp.com/) on Heroku. Your thoughts will be submitted to the creator's database.

![thoughtbox](https://github.com/eileenlimur/thoughtbox/blob/master/client/public/ThotBox.png)

Please let me know if you notice any performance-effecting errors.

## Stack
- React.js (via create-react-app)
- JavaScript
- Express
- Node.js
- PostgreSQL (via Heroku)

## Instructions

1. run `npm install` to install project dependencies

2. run `npm start` to run the app in the development mode on [http://localhost:3000](http://localhost:3000).
